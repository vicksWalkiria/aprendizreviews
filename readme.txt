Un carrusel de reseñas personalizable, creado por Aprendiz de SEO. Añade y gestiona reseñas desde el admin y muéstralas con el shortcode [reviews].

== Description ==

**Aprendiz Reviews** te permite:

- Añadir reseñas manualmente desde el panel de WordPress.
- Validarlas antes de que se muestren públicamente.
- Mostrarlas en cualquier parte del sitio mediante el shortcode `[reviews]`.
- Personalizarlas con avatar, estrellas y diseño adaptable.

Este plugin es ideal para negocios que quieran mostrar testimonios sin depender de servicios externos.

== Installation ==

1. Sube la carpeta del plugin a `/wp-content/plugins/aprendiz_reviews/`
2. Activa el plugin desde el panel de administración de WordPress
3. Ve a **Añadir Reseña** o **Gestionar Reseñas** para comenzar
4. Usa el shortcode `[reviews]` en cualquier entrada, página o widget

== Frequently Asked Questions ==

= ¿Puedo enviar reseñas desde el frontend? =
De momento solo el admin puede añadir reseñas. Pero se puede ampliar fácilmente.

= ¿Puedo personalizar el diseño? =
Sí. El plugin añade estilos base pero puedes sobrescribirlos con CSS en tu tema.
